Documentation

Next.js compatibility: full support for 13.x-16.x.

Features

  • Standalone mode — builds your Next.js app in standalone output mode for minimal deployment artifacts.
  • ISR with YDB — Incremental Static Regeneration backed by YDB for on-demand revalidation, tag-based invalidation, and path-based refresh.
  • Image optimization — automatic image optimization pipeline deployed alongside your application.
  • Middleware emulation — edge middleware runs in an emulated mode on Yandex Cloud Functions.
  • Server Actions — full support for React Server Actions with serverless execution.
  • Embedded Terraform — infrastructure is managed via embedded Terraform definitions, no manual IaC setup required.
